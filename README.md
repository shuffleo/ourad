# 《我地廣告》研究用廣告影片資料庫  
*A Research Archive of Hong Kong Television Commercials (1980s–2000s)*

## 簡介 Introduction

本資源庫 為《我地廣告》一書的**研究與教學輔助資料庫**，收錄多段香港八十年代至千禧年代初的電視廣告影片，用以配合書中對廣告創意、文化語境及時代關係的分析。

《我地廣告》聚焦九十年代香港電視廣告，從「廣告創作者」、「廣告業內人」、「研究者」及「香港人」等多重角度出發，探討廣告如何回應社會變化、文化身份與時代精神。本 資源庫 中的影片，作為分析對象與研究文本，協助讀者更準確地理解相關討論。

---

## 使用目的 Purpose of Use

本 資源庫 僅供以下用途：

- 學術研究  
- 教學與課堂討論  
- 文化與媒體研究  
- 配合《我地廣告》的閱讀與分析  

所有內容**不作任何商業用途**。
